# Nautilus-wallpaper
Generates a wallpaper with stats from a discord server

## wallpaper.sh
Generates and changes your wallpaper using feh

## Getting Started
### Setting up
* Get a discord bot Token and place it into a text file named `auth`
* run `sh wallpaper.sh`
